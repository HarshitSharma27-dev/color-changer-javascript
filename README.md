# JavaScript Color Changer

This is a beginner-friendly JavaScript project built to practice DOM Manipulation, Event Handling, and Dynamic UI Updates.

## Features

- Four color buttons
- Background color changes on click
- Displays currently selected color
- Responsive and clean UI

## Concepts Practiced

- DOM Manipulation
- Element Selection
- Event Listeners
- Click Events
- Style Manipulation
- Text Manipulation

## Technologies

- HTML5
- CSS3
- JavaScript

## How to Run

1. Clone the repository
2. Open index.html in your browser
